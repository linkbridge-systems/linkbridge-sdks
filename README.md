# LinkBridge SDKs

Official client libraries for the [LinkBridge](https://linkbridge.ng) e-invoicing API
(FIRS SI + APP — Nigeria).

> **This repository is a read-only mirror.** The source of truth lives in the private
> LinkBridge monorepo and is published here automatically on each release. File issues and
> feature requests at <https://github.com/linkbridge-systems/linkbridge-sdks/issues>.

| SDK | Install | Source |
|---|---|---|
| **Go** | `go get github.com/linkbridge-systems/linkbridge-sdks/sdk-go` | [`sdk-go/`](sdk-go) |
| **Node / TypeScript** | `npm install @linkbridge/sdk` | [`sdk-node/`](sdk-node) |
| **Python** | `pip install linkbridge` | [`sdk-python/`](sdk-python) |
| **PHP** | `composer require linkbridge/sdk-php` | [`sdk-php/`](sdk-php) |
| **Java** | Maven Central: `ng.linkbridge:linkbridge-sdk` — publishing setup in progress; consume via the [`sdk-java/`](sdk-java) sources in this repo until the first Central release | [`sdk-java/`](sdk-java) |

- **Docs & API reference:** <https://docs.linkbridge.ng>
- **Dashboard (sign up, get API keys):** <https://app.linkbridge.ng>
